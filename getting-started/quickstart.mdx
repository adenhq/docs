By the end of this guide, you will have Hive installed and run your first goal-driven agent.

## Prerequisites

<CardGroup cols={2}>
  <Card title="Python" icon="python">
    Python 3.11 or higher (3.11, 3.12, or 3.13)
  </Card>
  <Card title="LLM Provider" icon="key">
    API key for Anthropic, OpenAI, Gemini, or another supported provider
  </Card>
</CardGroup>

## Step 1: Clone the Repository

```bash
git clone https://github.com/adenhq/hive.git
cd hive
```

## Step 2: Run the Setup Wizard

Run the onboarding script:

```bash
./quickstart.sh
```

This will:

- Check Python and install `uv` if needed
- Install workspace packages (`framework` and `aden_tools`)
- Install Playwright browser dependencies when available
- Verify imports and local environment health
- Guide LLM provider configuration

<Accordion title="Manual Setup (Alternative)">
If you prefer manual setup or the wizard fails:

**Install workspace dependencies:**
```bash
uv sync
```

**Verify imports:**
```bash
uv run python -c "import framework; print('framework OK')"
uv run python -c "import aden_tools; print('aden_tools OK')"
uv run python -c "import litellm; print('litellm OK')"
```
</Accordion>

## Step 3: Set Up Environment

Export at least one provider API key:

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

<Info>
`./quickstart.sh` can detect existing keys in your environment and generate a default Hive provider configuration.
</Info>

## Step 4: Run Your First Agent

From the project root:

```bash
# Validate agent structure
PYTHONPATH=exports uv run python -m your_agent_name validate

# Show agent information
PYTHONPATH=exports uv run python -m your_agent_name info
```

Run the agent with sample input:

```bash
PYTHONPATH=exports uv run python -m your_agent_name run --input '{
  "task": "Process this request and provide structured output."
}'
```

<Tip>
Use `--mock` flag to test agents without making LLM API calls:
```bash
PYTHONPATH=exports uv run python -m your_agent_name run --mock --input '{...}'
```
</Tip>

## Step 5: Build a New Agent

Use Claude Code agent skills:

```bash
claude> /hive
claude> /hive-test
```

## You're Ready

You now have a working Hive development environment for building, testing, and running agents.
