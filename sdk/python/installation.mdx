## Install the SDK

```bash
pip install aden-py python-dotenv
```

### With Provider Support

<CodeGroup>

```bash OpenAI
pip install aden-py[openai]
```

```bash Anthropic
pip install aden-py[anthropic]
```

```bash Google Gemini
pip install aden-py[gemini]
```

```bash All Providers
pip install aden-py[all]
```

</CodeGroup>

### Framework Integrations

<CodeGroup>

```bash PydanticAI
pip install aden-py[pydantic-ai]
```

```bash LiveKit Voice Agents
pip install aden-py[livekit]
```

</CodeGroup>

## Environment Setup

Add to your shell configuration (`~/.bashrc` or `~/.zshrc`):

```bash
export OPENAI_API_KEY=sk-xxx
export ADEN_API_URL=https://kube.acho.io
export ADEN_API_KEY=your-aden-api-key
```

Or create a `.env` file:

```bash
OPENAI_API_KEY=sk-xxx
ADEN_API_URL=https://kube.acho.io
ADEN_API_KEY=your-aden-api-key
```

<Info>
Don't have an Aden API key yet? You can still use the SDK locally with the console emitter for testing.
</Info>

## Requirements

| Requirement | Version |
|-------------|---------|
| Python | >= 3.10 |

### Optional Dependencies

| Package | Version | Required For |
|---------|---------|--------------|
| `openai` | >= 1.0.0 | OpenAI provider |
| `tiktoken` | >= 0.5.0 | OpenAI token counting |
| `anthropic` | >= 0.20.0 | Anthropic provider |
| `google-generativeai` | >= 0.4.0 | Gemini provider |
| `aiohttp` | >= 3.8.0 | Async control agent |

## Verify Installation

```python
import os
from dotenv import load_dotenv
load_dotenv()

from aden import instrument, MeterOptions, create_console_emitter, uninstrument

# Instrument the SDK
instrument(MeterOptions(
    emit_metric=create_console_emitter(pretty=True),
))

print("Instrumentation successful!")

# Clean up
uninstrument()
```

Run the test:

```bash
python test.py
```

```bash venv
.venv\Scripts\activate     # Windows

```

```bash Poetry
poetry add aden-py

```

```bash uv
uv pip install aden-py[all]
```

</CodeGroup>
